# MEHAK-STORE
 
 ## Description
 
MehakStore is a e-commerce website for buying Perfumes.

## Technologies used(for now)

* HTML
* CSS

## Functionalities

* Home Page
* Product Listing Page
* Wishlist page 
* cart page
* Authentication
  * User Login
  * User signup
  * Logout
* Filters for sorting products
* user-profile
* Address management
* Forgot password
